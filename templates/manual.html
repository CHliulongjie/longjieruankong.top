中国中学场馆预约系统

操作手册

系统概述

中国中学场馆预约系统是一个基于Web的预约管理平台，支持羽毛球、乒乓球、篮球、足球等多种体育场馆的预约管理。

系统安装要求

  • 程序包自带Python虚拟环境，无需单独安装Python
  • 操作系统：Windows 7/10/11, Linux (Ubuntu 18.04+, CentOS 7+), macOS 10.14+
  • 内存：至少 2GB RAM
  • 存储空间：至少 500MB 可用空间
  • 网络：需要网络连接以支持远程解密功能
  • 浏览器：支持现代Web标准的浏览器（Chrome, Firefox, Safari, Edge）

系统运行指南

Windows系统

  1. 打开命令提示符或PowerShell，切换到程序根目录
  2. 双击运行 start.bat 文件
  3. 系统启动后会自动打开控制面板页面
  4. 如需手动访问，打开浏览器访问 http://localhost:5000/control

Linux/macOS系统

  1. 打开终端，使用 cd 命令切换到程序根目录
  2. 给启动脚本添加执行权限：chmod +x start.sh
  3. 运行命令：./start.sh
  4. 系统启动后会自动打开控制面板页面
  5. 如需手动访问，打开浏览器访问 http://localhost:5000/control

通用运行方法

  1. 打开终端或命令提示符，切换到程序根目录
  2. 运行命令：python main.py

本地账户管理

系统用户信息存储在根目录的 users.xlsx 文件中，您可以直接编辑此文件来添加或修改用户账户。

添加新用户步骤：

  1. 关闭系统（确保系统未运行）
  2. 使用Excel或其他支持XLSX格式的软件打开 users.xlsx 文件
  3. 在数据表格中添加新行，按以下格式填写信息：

用户信息格式：

列名 说明 示例
username 用户名（登录账号） zhangsan
password 用户密码 123456

注意事项：

  • 用户名必须唯一，不能重复
  • 密码建议使用安全密码
  • 修改后保存文件，重新启动系统即可生效

预约记录查看

  1. 确保系统已成功启动
  2. 打开控制面板（通常在 http://localhost:5000/control）
  3. 使用管理员账户登录
  4. 在左侧导航栏选择"预约记录"或"数据管理"
  5. 可以查看各场馆的预约情况
  6. 支持按日期、场馆类型筛选预约记录
  7. 可以导出预约数据为Excel格式

数据存储位置

系统的所有数据文件均存储在程序根目录的 data 文件夹中：

./data/

此目录包含各场馆的预约数据、用户信息等，建议定期备份此目录以确保数据安全。

常见错误与解决方法

1. 端口被占用错误

错误信息：Address already in use 或 端口占用相关错误

解决方法：

  • 检查是否有其他实例正在运行
  • 使用命令查找占用端口的进程：netstat -tulpn | grep :5000 (Linux) 或 netstat -ano | findstr :5000 (Windows)
  • 结束占用端口的进程后再启动系统

2. 依赖包缺失错误

错误信息：ModuleNotFoundError 或 ImportError

解决方法：

  • 安装依赖包：pip install -r requirements.txt
  • 如使用虚拟环境，请先激活虚拟环境再安装依赖

3. 远程解密失败

错误信息："远程解密失败" 或 "解密服务器不可用"

解决方法：

  • 检查网络连接是否正常
  • 确认远程解密服务器 https://longjieruankong.top 可访问
  • 检查防火墙设置，确保可以访问外部服务器

4. 权限不足错误

错误信息：Permission denied 或 权限相关错误

解决方法：

  • 以管理员权限运行（Windows）或使用sudo（Linux）
  • 检查项目目录的读写权限
  • 确保可以创建和修改数据文件

5. 文件编码错误

错误信息：UnicodeDecodeError 或 编码相关错误

解决方法：

  • 确保系统区域设置支持中文
  • 使用UTF-8编码保存配置文件

系统维护

日常维护

  • 定期备份数据文件（在 data/ 目录下）
  • 监控系统运行日志（在 logs/ 目录下）
  • 定期更新系统依赖以修复安全漏洞

系统更新

  1. 停止当前运行的系统
  2. 备份现有数据
  3. 替换更新文件
  4. 重启系统
