# 龙解软控 - Top 服务器

> **⚠️ 测试版本** - 此版本为测试版本，不代表最终结果。

## 项目简介

龙解软控 Top 服务器，包含文件加密系统和 web 服务。

## 当前版本修改内容

1. 迁移了 xyz 服务器的 web 服务到此仓库
2. 删除了招新网站相关代码
3. 集成了文件加密解密功能
4. 优化了路由结构

## 技术栈

- Python 3.x
- Flask
- SQLite
- Waitress

## 快速开始

```bash
pip install -r requirements.txt
python app.py
```

## 项目结构

```
├── app.py              # 主应用入口
├── templates/          # HTML 模板
├── static/             # 静态资源
└── README.md           # 项目说明
```
